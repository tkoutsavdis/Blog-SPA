# Responsive Blog SPA (Angular + Express + PostgreSQL)

A responsive single-page blog application built with **Angular** (frontend) and **Node.js/Express** (backend), backed by **PostgreSQL**.  
It supports dynamic post publishing, user accounts, and an admin dashboard for content and user management.

## Features
- Responsive SPA UI (mobile/tablet/desktop)
- Public blog feed + single post pages
- Create/edit/delete posts (admin)
- User authentication & authorization (roles)
- Admin dashboard for managing users and content
- PostgreSQL (posts, users, etc.)

## Tech Stack
- **Frontend:** Angular
- **Backend:** Node.js / Express
- **Database:** PostgreSQL

## Repository Structure
- `frontend/` (or your Angular folder) — Angular SPA
- `backend/` (or your Express folder) — Express API

## Notes
- `node_modules/` and build outputs are intentionally not included.
- `.env` files are intentionally not included.
- `Database` is empty.
